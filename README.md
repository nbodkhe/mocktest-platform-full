# MockTest Platform (FastAPI)

## Run (Docker)
1) Copy env  
   `cp .env.example .env`  (Windows: `copy .env.example .env`)
2) Start  
   `docker compose up --build`

- Docs: http://localhost:8080/docs  
- Health: http://localhost:8080/healthz  
- Ready: http://localhost:8080/readyz
